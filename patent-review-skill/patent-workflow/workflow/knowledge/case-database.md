## 1. 授权案例（正向）

### 案例1：深度神经网络模型的训练方法（审查示例5）

**来源**：《专利审查指南》（2024）第二部分第九章第6.2节

**技术方案**：
```
权利要求限定的解决方案涉及深度学习训练方法的改进，
针对不同大小的训练数据，选择适配具有不同处理效率的
单处理器训练方案或多处理器训练方案。
```

**三要素分析**：
| 要素 | 分析结论 |
|------|----------|
| 技术手段 | 算法与计算机系统内部结构存在特定技术关联 |
| 技术问题 | 解决训练速度慢的问题 |
| 技术效果 | 提升模型训练过程中硬件的执行效果 |

**判定结果**：属于专利法第二条第二款规定的技术方案

---

### 案例2：电子券使用倾向度的分析方法（审查示例6）

**来源**：《专利审查指南》（2024）第二部分第九章第6.2节

**技术方案**：
```
处理电子券相关的大数据，对电子券进行归类、获取样本数据、
确定行为特征并进行模型训练，挖掘用户行为特征与电子券
使用倾向度之间的内在关联关系。
```

**三要素分析**：
| 要素 | 分析结论 |
|------|----------|
| 技术手段 | 分类、聚类等算法处理具体应用领域数据 |
| 技术问题 | 提升分析用户对电子券使用倾向度的精确性 |
| 技术效果 | 挖掘的内在关联关系符合自然规律 |

**判定结果**：属于专利法第二条第二款规定的技术方案

---

### 案例3：知识图谱推理方法（审查示例7）

**来源**：《专利审查指南》（2024）第二部分第九章第6.2节

**技术方案**：
```

通过对问答系统、语义搜索中相关联的知识进行实体识别
和关系抽取构建知识图谱并进行知识图谱推理。
```

**三要素分析**：
| 要素 | 分析结论 |
|------|----------|
| 技术手段 | 实体识别、关系抽取等自然语言处理技术 |
| 技术问题 | 丰富语义信息和提高推理准确性 |
| 技术效果 | 文本嵌入及语义搜索效果提升 |

**判定结果**：属于专利法第二条第二款规定的技术方案

## 2. 不授权案例（反向）

### 案例5：金融产品价格预测方法（审查示例10）

**来源**：《专利审查指南》（2024）第二部分第九章第6.2节

**技术方案**：
```
利用神经网络模型挖掘历史金融产品的价格数据与未来价格
数据之间的内在关联关系。
```

**三要素分析**：
| 要素 | 分析结论 |
|------|----------|
| 技术手段 | 神经网络模型（通用算法） |
| 技术问题 | 如何预测金融产品价格 |
| 技术效果 | 历史价格高低不能决定未来价格走势 |

**不授权理由**：
- 金融产品价格走势遵循经济学规律，而非自然规律
- 历史价格与未来价格不存在符合自然规律的内在关联
- 解决的问题不构成技术问题
- 获得的效果不构成技术效果

**判定结果**：不属于专利法第二条第二款规定的技术方案

---

### 案例6：抽象算法（不属于任何技术领域）

**来源**：《专利审查指南》（2024）

**不授权情形**：
- 纯粹的数学模型建立方法
- 不包含任何技术特征的抽象算法
- 权利要求全部内容均为智力活动的规则和方法

**判定结果**：依据专利法第二十五条不授权

---

### 案例7：游戏规则/玩法

**来源**：专利审查实践

**典型示例**：
- 网络游戏玩法设计
- 棋牌游戏规则
- 竞赛评分方法

**不授权理由**：
- 不解决技术问题
- 不采用技术手段
- 不产生技术效果

**判定结果**：不属于专利保护的客体

---

### 案例8：商业返利规则

**来源**：专利审查实践

**不授权情形**：
```
一种根据用户的消费额度进行返利的方法，该方法中包含的
特征全部是与返利规则相关的商业规则和方法特征，不包含
任何技术特征。
```

**判定结果**：依据专利法第二十五条不授权

## 3. 边界案例分析

### 案例9：物流配送方法（审查示例13，修改后）

**来源**：《专利审查指南》（2024）第二部分第九章第6.2节

**特殊考量**：用户体验提升

**方案要点**：
```
- 数据架构和数据通信方式的调整
- 取件通知规则
- 批量通知实现方式
```

**用户体验提升认定**：
| 用户体验 | 提升来源 | 可否在创造性中考虑 |
|----------|----------|-------------------|
| 物流派送人员操作便利 | 技术特征 | 可考虑 |
| 订货用户接收取货通知及时 | 技术特征+算法特征 | 可考虑 |

**判定要点**：用户体验提升与技术特征相关联时，可在创造性审查中考虑

---

### 案例10：用于适配神经网络参数的方法（审查示例15）

**来源**：《专利审查指南》（2024）第二部分第九章第6.2节

**技术方案**：
```
将权重参数的尺寸填充为基于硬件使用率确定的、对应维度
上的目标尺寸，以便硬件高效处理数据。
```

**创造性分析**：
| 要素 | 分析结论 |
|------|----------|
| 算法与硬件关联 | 算法提升硬件运算效率 |
| 功能相互作用 | 算法特征与技术特征紧密关联 |
| 创造性贡献 | 应考虑算法对技术方案的贡献 |
